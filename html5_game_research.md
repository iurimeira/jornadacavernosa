# Pesquisa de Desenvolvimento de Jogos HTML5 para Modo Caverna

## Frameworks Analisados

### Phaser
- **Site oficial**: [https://phaser.io/](https://phaser.io/)
- **Descrição**: Framework completo para desenvolvimento de jogos HTML5 2D
- **Vantagens**:
  - Framework maduro e bem estabelecido
  - Suporte a tilemaps via Tiled
  - Suporte a animações via Spine
  - Comunidade ativa e muitos tutoriais disponíveis
  - Tutoriais específicos para jogos estilo Zelda
  - Suporte a física, colisões, entrada de usuário e áudio
  - Documentação extensa
- **Recursos relevantes**:
  - Tutoriais de jogos estilo Zelda com Phaser 3
  - Suporte a tilemaps para criação de mundos modulares
  - Sistema de diálogo pode ser implementado

### PixiJS
- **Site oficial**: [https://pixijs.com/](https://pixijs.com/)
- **Descrição**: Renderizador WebGL 2D rápido e flexível
- **Vantagens**:
  - Foco em performance de renderização
  - Mais flexível, mas menos recursos específicos para jogos
  - Bom para projetos que precisam de controle mais detalhado
- **Desvantagens**:
  - Menos recursos específicos para jogos comparado ao Phaser
  - Requer mais código para implementar mecânicas de jogo

## Recursos de Arte

### OpenGameArt.org
- **Site oficial**: [https://opengameart.org/](https://opengameart.org/)
- **Descrição**: Repositório de assets gratuitos para jogos
- **Recursos relevantes**:
  - Sprites 16x16 para RPGs
  - Tilesets para mapas
  - Elementos de UI retrô
  - Efeitos sonoros e música

### Tutoriais de Jogos Estilo Zelda com Phaser
- Série "How To Make A Zelda-Like Game With Phaser 3"
- Tutoriais sobre implementação de tilemaps em Phaser 3
- Tutoriais sobre sistemas de diálogo

## Conclusão da Pesquisa

Após analisar as opções disponíveis, o **Phaser** parece ser a escolha mais adequada para o desenvolvimento do jogo Modo Caverna pelos seguintes motivos:

1. Possui recursos específicos para jogos que facilitarão o desenvolvimento
2. Tem suporte nativo a tilemaps via Tiled, essencial para criar o mundo do jogo
3. Existem tutoriais específicos para jogos estilo Zelda usando Phaser
4. A comunidade ativa e documentação extensa facilitarão a resolução de problemas
5. O framework é maduro e estável, reduzindo riscos técnicos
